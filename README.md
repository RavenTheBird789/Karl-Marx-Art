The file containted within this repository is a python program using ANSI colors to display an art piece showcasing the sickle and hammer 
along with a quote attributed to Karl Mark by several secondary sources

To run, simply type "git clone https://github.com/RavenTheBird789/Karl-Marx-Art" in your terminal and "python3 marx_quote_art.py" and the art work will be displayed on your screen
